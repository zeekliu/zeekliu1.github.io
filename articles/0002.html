如何制作简单的StudentMainKiller

建议使用 JiYuTrainer （JiYuTrainer.exe）

警告⚠：上课掐极域有风险，需小心谨慎

或许最简单的选择是studentmainkiller.bat，着实方便小巧：

taskkill /im StudentMain.exe /f

可是这样使用不是很自在，无法做到“想听就听”的境界，一掐就全没了，老师控屏时又不能随时关闭。

所以我们需要用更高级的代码：按下特定的键就会立刻taskkill

正好，在字母键中极域除了b、c其他字母键全被封了，所以程序设置就是：按下b键和c键就杀极域！

于是只要检测b、c的按下（一定要全局，所以c++的getch无效），就立即kill

附上Python代码（为什么用Python？因为Python方便，就8行）：

import keyboard
from os import system
print("Press key B and C to kill StudentMain.exe")
while True:
    while not (keyboard.is_pressed('c') and keyboard.is_pressed('b')):
        pass
    while keyboard.is_pressed('c') and keyboard.is_pressed('b'):
        system("taskkill /im StudentMain.exe /f")

注意：运行本代码需要keyboard库，可以通过 pip install keyboard 下载
